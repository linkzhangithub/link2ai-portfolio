项目作品

AI写作助手

项目背景：长文写作场景下，用户常面临大纲构思困难、内容扩写耗时的问题。本项目通过AI辅助，实现从大纲生成到分节扩写的完整写作流程。

核心功能：输入主题自动生成结构化大纲，支持章节调整；分节内容一键生成，支持润色、扩写、缩写三种改写模式；打字机效果逐字展示，支持随时中断生成。

技术亮点：AbortController实现请求中断控制，优化用户交互体验；大纲驱动的生成策略，确保内容结构一致性。

技术栈：Vue3 · Pinia · Vite · Node.js · Express · LLM API集成

RAG知识库问答系统

项目背景：企业内部文档分散、检索效率低，传统关键词搜索难以理解语义。本项目基于RAG技术，实现基于文档内容的精准问答。

核心功能：支持PDF/TXT/MD/DOCX多格式文档上传与解析；混合检索策略：向量相似度60% + 关键词匹配40%；两阶段重排序：Top-20候选 + 关键词覆盖度评分；阈值过滤机制，低于阈值的结果不返回。

技术亮点：混合检索兼顾语义理解与关键词匹配，提升召回准确率；文档分块策略：多分隔符优先 + 固定长度 + 重叠窗口。

技术栈：Vue3 · Node.js · Express · 向量检索 · LLM API集成 · Docker

AI对话机器人

项目背景：通用AI对话缺乏场景适配，用户在不同场景下需要不同风格的回复。本项目通过预设角色和语音交互，提升对话体验。

核心功能：预设4种角色：智能助手、前端面试官、心理导师、贴吧老哥；SSE流式输出实现打字机效果，减少等待感；语音输入转文字，支持实时语音转写；图片理解功能，支持上传图片提问。

技术亮点：SSE流式输出，首字节响应时间<500ms；角色SystemPrompt设计，确保回复风格一致性。

技术栈：Vue3 · Pinia · SSE流式输出 · LLM API集成 · 语音识别SDK · 图片理解
