项目作品

RAG知识库问答系统

项目背景：企业内部文档分散、传统关键词搜索难以理解语义。本项目基于RAG技术，实现基于文档内容的精准问答。

核心功能：支持PDF/TXT/MD/DOCX多格式文档上传；混合检索策略提升召回准确率；两阶段重排序优化检索结果。

技术栈：Vue3 · Vue Router · Node.js · Express · 向量检索 · 限流控制 · LLM API集成 · Docker

AI写作助手

项目背景：长文写作场景下，用户常面临大纲构思困难、内容扩写耗时的问题。本项目通过AI辅助，实现从大纲生成到分节扩写的完整写作流程。

核心功能：输入主题自动生成结构化大纲；分节内容一键生成，支持润色、扩写、缩写；打字机效果展示，支持随时中断。

技术栈：Vue3 · Pinia · Vite · Tailwind CSS · Node.js · Express · LLM API集成

AI对话机器人

项目背景：通用AI对话缺乏场景适配，用户在不同场景下需要不同风格的回复。本项目通过预设角色和语音交互，提升对话体验。

核心功能：预设4种角色适配不同场景；SSE流式输出减少等待感；语音输入转文字；图片理解功能。

技术栈：Vue3 · Vue Router · Pinia · SSE流式输出 · Markdown渲染 · 代码高亮 · 图片压缩 · LLM API集成 · 语音识别SDK · 图片理解
