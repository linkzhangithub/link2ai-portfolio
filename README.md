# Link Zhang's Portfolio

个人作品集网站，展示前端开发和AIGC应用项目。

## 项目简介

这是我的个人作品集网站，用于展示我的前端开发技能和AIGC相关项目。网站采用纯HTML/CSS/JavaScript实现，响应式设计，适配不同设备。

## 技术栈

- **前端**：HTML5, CSS3, JavaScript
- **样式**：CSS Variables, Flexbox, Grid
- **图标**：Font Awesome
- **部署**：EdgeOne Pages

## 项目展示

- **AI 对话助手**：基于 Vue 3 + Pinia + Vite + Node.js + 智谱 AI (GLM-4) + 讯飞语音构建的智能对话系统
- **RAG 智能助手**：基于检索增强生成技术的智能问答系统
- **智能简历分析系统**：基于AI技术的简历分析工具

## 部署

网站部署在 EdgeOne Pages，通过 GitHub 仓库自动部署。
