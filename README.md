# Link Zhang's Portfolio

个人作品集网站，展示前端开发和AI应用项目。

## 项目简介

这是我的个人作品集网站，展示我的前端开发技能和AI相关项目。

## 技术栈

- **前端**：HTML5, CSS3, JavaScript
- **样式**：CSS Variables, Flexbox, Grid
- **图标**：Font Awesome
- **动画**：ScrollReveal, vanilla-tilt

## 项目展示

- **RAG知识库问答系统**：支持PDF/TXT/MD/DOCX多格式文档上传，采用混合检索策略（向量相似度+关键词匹配），实现精准的文档问答
- **AI写作助手**：基于大纲驱动的长文生成工具，支持分节扩写、内容润色和缩写，实现请求中断控制
- **AI对话机器人**：支持多轮对话与语音输入，预设4种角色（智能助手、前端面试官、心理导师、贴吧老哥），采用SSE流式输出实现打字机效果

## 本地预览

由于这是一个纯静态网站，你可以通过以下方式在本地预览：

**方法 1：使用 VS Code Live Server（推荐）**
1. 安装 VS Code 的 "Live Server" 扩展
2. 右键点击 `index.html`，选择 "Open with Live Server"

**方法 2：使用 Python**
```bash
python -m http.server 8000
```

**方法 3：使用 Node.js**
```bash
node server.js
```
